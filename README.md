# ЁЯЪА 15-Week Web Development Bootcamp

Welcome to this intensive, **beginner-to-advanced** web development bootcamp! This course is designed to give you hands-on skills in HTML, CSS, JavaScript, frameworks, and backend development.

### ЁЯЧУя╕П Course Schedule: 15 Weeks, 30 Tasks

### ЁЯУН Progress from **HTML Basics** to a **Full-Stack Project**

---

## ЁЯУЪ **Course Outline**

### Week 1: HTML Basics ЁЯМР

* **Task 1:** **Learn HTML Basics**
  ЁЯУД Study HTML structure, elements, and attributes with [MDN&#39;s HTML Guide](https://developer.mozilla.org/en-US/docs/Web/HTML) or [W3Schools]().

  ЁЯХТ Due in: **3 days**
* **Task 2:** **Build a Simple Personal Webpage**
  ЁЯЫая╕П Create a basic webpage with your bio, skills, and contact info.

  ЁЯХТ Due in: **9 days**

---

### Week 2: CSS Basics ЁЯОи

* **Task 1:** **Learn CSS Basics**
  ЁЯФН Understand CSS selectors, properties, and the box model with [MDN&#39;s CSS Guide](https://developer.mozilla.org/en-US/docs/Web/CSS) or [CSS Basics on W3Schools]().

  ЁЯХТ Due in: **10 days**
* **Task 2:** **Style Your Personal Webpage with CSS**
  ЁЯОи Add styles to your webpage to make it look amazing!

  ЁЯХТ Due in: **16 days**

---

### Week 3: Responsive Design ЁЯУ▒

* **Task 1:** **Learn Responsive Design Techniques**
  ЁЯУП Study media queries, Flexbox, and CSS Grid. Great resources include [A Complete Guide to Flexbox]() and [CSS Grid Layout Guide]().

  ЁЯХТ Due in: **17 days**
* **Task 2:** **Create a Responsive Layout**
  ЁЯУ▒ Build a multi-section webpage that adapts to different screen sizes.

  ЁЯХТ Due in: **23 days**

---

### Week 4: JavaScript Basics ЁЯТ╗

* **Task 1:** **Learn JavaScript Basics**
  ЁЯУЦ Get familiar with variables, data types, functions, and control structures with [JavaScript on MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript).

  ЁЯХТ Due in: **24 days**
* **Task 2:** **Add Interactivity with JavaScript**
  ЁЯОЙ Add basic interactivity like buttons and forms to your webpage.

  ЁЯХТ Due in: **30 days**

---

### Week 5: DOM Manipulation ЁЯУЭ

* **Task 1:** **Learn DOM Manipulation**
  ЁЯФО Access and modify HTML elements using JavaScript. Start with [JavaScript DOM on W3Schools]().

  ЁЯХТ Due in: **31 days**
* **Task 2:** **Build a To-Do List App**
  тЬФя╕П Create an app to add, remove, and mark tasks as complete.

  ЁЯХТ Due in: **37 days**

---

### Week 6: Version Control (Git) ЁЯЧГя╕П

* **Task 1:** **Learn Git Basics**
  ЁЯУМ Learn Git commands, branching, and merging. Start with [Git Basics on GitHub Docs](https://docs.github.com/en/get-started/using-git).

  ЁЯХТ Due in: **38 days**
* **Task 2:** **Set up a GitHub Repository**
  ЁЯЫая╕П Create a GitHub account and set up a repository.

  ЁЯХТ Due in: **44 days**

---

### Week 7: Advanced CSS (Animations) ЁЯОЮя╕П

* **Task 1:** **Learn CSS Animations**
  ЁЯХ╣я╕П Study keyframes, transitions, and animations with [CSS Animations on MDN](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Animations/Using_CSS_animations).

  ЁЯХТ Due in: **45 days**
* **Task 2:** **Create a CSS Animation Project**
  тЬи Build a landing page with animated elements to showcase your CSS skills.

  ЁЯХТ Due in: **51 days**

---

### Week 8: JavaScript ES6+ Features ЁЯЪА

* **Task 1:** **Learn JavaScript ES6+**
  ЁЯУЬ Explore features like arrow functions, destructuring, and modules. Learn more on [JavaScript ES6 on MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference).

  ЁЯХТ Due in: **52 days**
* **Task 2:** **Refactor Your To-Do List App with ES6**
  ЁЯФД Update your app with ES6 syntax for cleaner code.

  ЁЯХТ Due in: **58 days**

---

### Week 9: Introduction to Frameworks (React/Vue) тЪЫя╕П

* **Task 1:** **Learn React or Vue**
  ЁЯзй Study components, state, and hooks in [React]() or [Vue]().

  ЁЯХТ Due in: **59 days**
* **Task 2:** **Build a Simple App**
  ЁЯЫая╕П Use React or Vue to create a small application to practice.

  ЁЯХТ Due in: **65 days**

---

### Week 10: API Basics (REST) ЁЯФЧ

* **Task 1:** **Learn API Basics**
  ЁЯМР Understand RESTful APIs and making HTTP requests. Start with [APIs on MDN](https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Client-side_web_APIs/Introduction).

  ЁЯХТ Due in: **66 days**
* **Task 2:** **Create a Weather App**
  ЁЯМжя╕П Use a public weather API to display data in a weather app.

  ЁЯХТ Due in: **72 days**

---

### Week 11: CSS Preprocessors (Sass) ЁЯОи

* **Task 1:** **Learn Sass Basics**
  ЁЯОи Study Sass with variables, nesting, and mixins. Check [Sass Basics]().

  ЁЯХТ Due in: **73 days**
* **Task 2:** **Refactor Your CSS with Sass**
  ЁЯУЛ Update your CSS for better organization with Sass.

  ЁЯХТ Due in: **79 days**

---

### Week 12: Build Tools (Webpack) ЁЯЫая╕П

* **Task 1:** **Learn Webpack Basics**
  ЁЯФз Set up and configure Webpack for asset management with [Webpack Docs]().

  ЁЯХТ Due in: **80 days**
* **Task 2:** **Set Up a Project with Webpack**
  ЁЯПЧя╕П Create a project with Webpack to manage dependencies.

  ЁЯХТ Due in: **86 days**

---

### Week 13: Introduction to Backend (Node.js) ЁЯЦея╕П

* **Task 1:** **Learn Node.js Basics**
  ЁЯЦея╕П Study Node.js, including modules and npm, with [Node.js Docs]().

  ЁЯХТ Due in: **87 days**
* **Task 2:** **Build a REST API with Node.js**
  ЁЯФМ Create a basic REST API using Node.js and Express.

  ЁЯХТ Due in: **93 days**

---

### Week 14: Database Basics (MongoDB) ЁЯЧДя╕П

* **Task 1:** **Learn MongoDB Basics**
  ЁЯЧДя╕П Understand MongoDB collections and documents with [MongoDB Documentation](https://www.mongodb.com/docs/).

  ЁЯХТ Due in: **94 days**
* **Task 2:** **Connect Node.js API to MongoDB**
  ЁЯФЧ Integrate MongoDB for storing and retrieving data.

  ЁЯХТ Due in: **100 days**

---

### Week 15: Final Project ЁЯМЯ

* **Task:** **Build a Full-Stack Application**
  ЁЯЫая╕П Combine your front-end and backend skills into a final project.

  ЁЯХТ Due in: **101 days**
